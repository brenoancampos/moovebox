# LED Floors — Landing Page

Landing page premium para **venda de Pistas de LED para eventos**, com foco em conversão: posiciona a pista como uma fonte de renda recorrente (locação para casamentos, 15 anos, formaturas e eventos corporativos).

Feita em **HTML, CSS e JavaScript puro**, sem frameworks nem bibliotecas externas.

## Arquivos

| Arquivo | Descrição |
| --- | --- |
| `index.html` | Estrutura e conteúdo de todas as seções |
| `style.css` | Visual premium (fundo preto, dourado/azul neon, glassmorphism, animações) |
| `script.js` | Calculadora de lucro, gráficos em canvas, carrossel, animações de scroll |

## Seções

Cabeçalho fixo · Hero · Produtos (Paris e Galáxia) · Comparativo · Calculadora de Lucro com gráficos · Por que investir · Timeline · Galeria · Vídeos · Depoimentos · FAQ · Garantia · Chamada final · Rodapé.

## Como rodar

Site estático: basta abrir `index.html` no navegador. Sem build, sem dependências.

## O que personalizar

Procure no código pelos comentários `INSERIR:` — eles marcam onde colocar logo, fotos das pistas, vídeos, depoimentos, dados de contato e redes sociais.
